# v0.9.0 — Console: coordinate sessions Pallium owns

Pallium can now spawn, watch, and manage its own coding sessions.

## What's new

- **`pallium console`.** A coordination surface for live session visibility,
  handoffs between agents, file claims, and review gates.
- **Spawn owned sessions.** `console run` starts a session Pallium tracks,
  either in the foreground or detached in the background.
- **Read and manage owned sessions.** `console read` retrieves captured logs;
  `console owned list`/`show` inspect session metadata; `console interrupt`
  stops one cleanly.

## Boundary

Pallium controls sessions it spawns itself. It does not inject commands into
sessions it didn't start.

## Install

```bash
npm install -g pallium
pallium version
```
