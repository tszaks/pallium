# v0.8.3 — More robust transcript parsing

A follow-up fix for the same class of large-transcript indexing issue.

## Fixed

- **Switched to a more robust parser.** Session indexing now uses a reader
  that isn't limited by a fixed buffer size, closing the remaining large-event
  parsing failures from the previous release.

## Install

```bash
npm install -g pallium
pallium version
```
