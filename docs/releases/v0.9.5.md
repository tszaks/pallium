# v0.9.5 — Gates become autonomous verifiers

A patch release for repo-memory accuracy, plus a real change to how workflow
gates behave.

## What's new

- **Gates are now autonomous.** `gate()` no longer pauses a workflow for a
  human to approve — it runs a verifier agent that approves or rejects
  automatically, so a workflow can enforce quality checks without stopping
  to wait on someone.
- **Continuous integration.** Every push and pull request now runs the full
  test suite automatically.

## Fixed

- **More accurate git history parsing.** Multi-line commit messages could
  previously be mistaken for changed file paths. Fixed, with regression
  coverage.

## Install

```bash
npm install -g pallium
pallium version
```
