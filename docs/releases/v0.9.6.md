# v0.9.6 — Hardening for open source

Prepares Pallium's workflow engine and the repository itself for wider open
source use.

## What's new

- **More reliable resume.** Workflow runs resume more predictably: cached
  results, budgets, and pending decisions all survive a pause and resume
  correctly.
- **Safer parallel execution.** A failure in one parallel worker no longer
  risks affecting its siblings.
- **Secured the HTTP API.** `pallium workflow serve` now requires a bearer
  token for any non-local connection.
- **Real open source infrastructure.** Added security scanning, dependency
  updates, issue and PR templates, a code of conduct, and a security policy.

## Install

```bash
npm install -g pallium
pallium version
```
