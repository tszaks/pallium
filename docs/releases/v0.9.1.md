# v0.9.1 — Pallium comes to npm

Pallium is now installable with a single npm command.

## What's new

- **Published to npm.** `npm install -g pallium` is now the primary way to
  install Pallium.
- **Prebuilt binaries.** Release automation now builds and attaches macOS and
  Linux binaries (arm64 and x64) with checksums for verification.
- **No-registry fallback.** A downloadable npm tarball is attached to every
  release for environments that can't reach the npm registry.

## Install

```bash
npm install -g pallium
```

No-registry fallback:

```bash
npm install -g https://github.com/tszaks/pallium/releases/download/v0.9.1/pallium-0.9.1.tgz
```

Or with Go:

```bash
go install github.com/tszaks/pallium@latest
```
