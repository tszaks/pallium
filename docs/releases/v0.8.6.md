# v0.8.6 — More headroom for large local archives

Continues hardening session indexing for people with large local Codex
transcript archives.

## Fixed

- **Lower memory ceiling across the board.** Capped per-message text and
  search data sizes and simplified redaction handling, so `sessions index`
  completes reliably on large local archives instead of running out of
  memory.

## Install

```bash
npm install -g pallium
pallium version
```
