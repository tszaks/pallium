# v0.7.7 — Know when guidance is stale

Pallium now tells you not just what it found, but how fresh and well-evidenced
that information is.

## What's new

- **Freshness metadata.** Every report now says whether the guidance behind it
  might be stale, so an agent can decide whether to trust it or re-check.
- **Evidence metadata.** Reports show what each piece of guidance was actually
  based on.
- **Visible in the CLI.** Freshness and evidence now show up in Pallium's
  normal human-readable output, not just in `--json`.

## Install

```bash
npm install -g pallium
pallium version
```
