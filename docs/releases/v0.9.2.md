# v0.9.2

Pallium v0.9.2 makes npm the primary install path and documents the published
package clearly in the repo.

## Added

- Published npm install instructions to the README.
- Added direct GitHub release tarball install instructions for environments
  where npm registry lookup is blocked.
- Attached the packed npm package to the GitHub release alongside native
  binaries and checksums.

## Install

```bash
npm install -g pallium
pallium version
```

No-registry fallback:

```bash
npm install -g https://github.com/tszaks/pallium/releases/download/v0.9.2/pallium-0.9.2.tgz
```
