# v0.9.2 — npm is now the primary install path

A documentation and distribution cleanup release.

## What's new

- **Clearer install instructions.** The README now leads with the npm install
  path.
- **Direct download fallback.** A packaged release tarball is attached to
  every GitHub release for environments where the npm registry isn't
  reachable.

## Install

```bash
npm install -g pallium
pallium version
```

No-registry fallback:

```bash
npm install -g https://github.com/tszaks/pallium/releases/download/v0.9.2/pallium-0.9.2.tgz
```
