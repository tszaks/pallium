# v0.9.12 — Session-scoping fixes

Four real bugs, all surfaced by everyday use rather than testing, all fixed.

## Fixed

- **No more cross-branch task confusion.** A repo checkout shared across
  branches could leak an unrelated task into the wrong branch's results.
  Fixed.
- **Cleaner run reports.** Workflow reports no longer bundle in unrelated
  session history from other repos by default.
- **Honest status on partial failure.** A run where some workers finished
  before the run itself failed now reports that clearly instead of showing a
  flat, sometimes-blank failure message.
- **`workflow generate --cwd`.** Generating a workflow for another repo now
  works reliably without having to change directories first.

## Install

```bash
npm install -g pallium
pallium version
```
