# v0.8.4 — Cap memory use on huge sessions

Keeps session indexing from ballooning memory and disk use on unusually
large Codex transcripts.

## Fixed

- **Bounded storage for huge events.** Raw event data is now truncated before
  storage while still preserving parsed messages, metadata, and search
  results in full.

## Install

```bash
npm install -g pallium
pallium version
```
