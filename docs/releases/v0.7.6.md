# v0.7.6 — Better import resolution

Pallium gets sharper at tracing how files actually connect to each other
across TypeScript and Python projects.

## What's new

- **TypeScript and JavaScript path aliases.** Pallium now resolves aliases
  defined in the nearest `tsconfig.json` or `jsconfig.json`, including configs
  inherited from a parent project.
- **Python `src`-layout support.** Import resolution now handles the common
  `src/`-based package layout correctly.

## Install

```bash
npm install -g pallium
pallium version
```
