# v0.9.4 — Dynamic workflows

The biggest addition yet: Pallium can now run multi-agent orchestration as
real, executable code.

## What's new

- **Dynamic workflows.** Write orchestration as an async JavaScript script —
  fan out parallel workers, stream work through pipeline stages, gate on
  verification — and let Pallium run it as a durable, resumable process.
- **Streaming pipelines.** `pipeline(items, stage1, stage2, ...)` lets fast
  items move ahead instead of waiting behind slower ones in the same batch.
- **Safer by default.** Workflow scripts get guardrails against non-
  deterministic code and runaway loops, so resumed runs stay reliable and
  bounded.

## Install

```bash
npm install -g pallium
pallium version
```
