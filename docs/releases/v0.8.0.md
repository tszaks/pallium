# v0.8.0 — Session memory arrives

Pallium can now remember your past Codex sessions, not just your code.

## What's new

- **Session memory.** Pallium indexes your Codex CLI transcripts so past
  sessions become searchable, structured history instead of scrollback you
  have to hunt through by hand.
- **Live session visibility.** See in-progress sessions as they happen, not
  just after they end.
- **Full-text and semantic search.** Search past sessions by keyword, grep
  raw transcripts, or search by meaning using OpenAI embeddings.
- **Session stats.** A quick overview of your indexed session history.

## Install

```bash
npm install -g pallium
pallium version
```
