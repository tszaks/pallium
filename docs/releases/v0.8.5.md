# v0.8.5 — Tighter caps on raw session storage

Further tightens the memory and storage caps from the previous release.

## Fixed

- **Lower storage ceiling for raw events.** Session indexing keeps focusing
  on searchable messages and results, with a firmer cap on raw event storage
  so large local transcript archives don't grow unbounded.

## Install

```bash
npm install -g pallium
pallium version
```
