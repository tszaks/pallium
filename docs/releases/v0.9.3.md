# v0.9.3 — GitHub Packages mirror

Adds a second install path for teams that prefer GitHub's own package
registry.

## What's new

- **GitHub Packages support.** Pallium is now also published as
  `@tszaks/pallium` on GitHub Packages, alongside the primary npm package.

## Install

Primary:

```bash
npm install -g pallium
```

GitHub Packages mirror:

```bash
npm config set @tszaks:registry https://npm.pkg.github.com
npm install -g @tszaks/pallium
```
