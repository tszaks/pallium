# v0.9.3

Pallium v0.9.3 adds a GitHub Packages npm distribution while keeping the public
npm package as the primary install path.

## Added

- Publish support for the GitHub Packages npm package `@tszaks/pallium`.
- A workflow to publish the scoped GitHub package with `GITHUB_TOKEN`.
- A local package-prep script that builds the scoped GitHub package from the
  same npm wrapper used by the public `pallium` package.
- README instructions for installing from GitHub Packages.

## Install

Primary npm package:

```bash
npm install -g pallium
```

GitHub Packages mirror:

```bash
npm config set @tszaks:registry https://npm.pkg.github.com
npm install -g @tszaks/pallium
```
