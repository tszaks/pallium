# Code of Conduct

## Our Standard

This project expects respectful, practical collaboration. Contributors should focus on the work, provide evidence for technical claims, and avoid personal attacks, harassment, or discriminatory language.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment or abuse
- Personal insults
- Publishing private information without permission
- Spam or repeated off-topic comments
- Deliberately disruptive conduct

## Enforcement

Maintainers may edit, hide, or remove comments, close issues, restrict participation, or block users when needed to keep the project useful and safe.

Report conduct issues through GitHub private contact paths where possible. Do not include sensitive personal information in public issues.
